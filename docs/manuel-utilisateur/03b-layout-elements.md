# Manuel utilisateur - Layout libre : Éléments et presets

Guide détaillé de chaque type d'élément disponible dans la bibliothèque du Layout libre.

## Éléments de match

- **Score** : affiche le score d'une équipe. Sélecteur Côté (Gauche/Droite)
- **Nom d'équipe** : nom + drapeau optionnel. Sélecteur Côté
- **Drapeau** : drapeau seul (sans texte). Sélecteur Côté
- **Horloge** : temps du match. Options : afficher période, afficher cadre
- **Période** : indicateur de période en cours
- **Temps morts** : indicateurs visuels des deux équipes
- **Tirs au but** : tentatives des deux équipes (cercles vert/rouge/gris)

## Éléments texte et données

- **Bloc de texte** : texte libre avec police, taille, épaisseur, alignement, casse, espacement
- **Ligne de stat** : valeur gauche + libellé + valeur droite
- **Barre comparative** : barre de comparaison entre deux valeurs

## Éléments joueurs et médias

- **Photo joueur** : photo avec forme cercle ou carré
- **Image** : image externe avec ajustement (couvrir, contenir, étirer)
- **Forme** : rectangle, cercle ou rectangle arrondi avec couleur et bordure
- **Séparateur** : ligne horizontale ou verticale

## Éléments composés

- **Header complet** : bloc automatique avec drapeaux, noms, scores, temps morts, tirs au but
- **Colonne de pénalités** : liste des pénalités d'une équipe
- **Types d'affichage embarqués** : les 13 types classiques utilisables comme éléments

## Gestion des couches

Onglet Calques du rail : liste de tous les éléments, sélection, contrôle du z-index via 4 boutons (Premier plan, Avancer, Reculer, Arrière-plan).

## Presets

Onglet Presets du rail : sauvegarder un champ individuel, sauvegarder l'écran complet, charger un preset.
