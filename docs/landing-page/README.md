Scoreboard Design est un studio de création broadcast complet, conçu pour les professionnels du design visuel et de l'événementiel sportif. Il réunit dans une seule application tout ce qu'il faut pour concevoir, personnaliser et exploiter en direct des habillages de scores destinés à l'incrustation vidéo — sans jamais sacrifier la liberté créative ni la fiabilité en conditions live.

---

## En résumé

| Capacité | Détail |
|----------|--------|
| Types d'affichage | 16 (dont Layout libre avec 45 éléments) |
| Canaux de couleur | 14, chacun avec opacité indépendante |
| Presets de couleurs | 5 thèmes professionnels |
| Polices broadcast | 25, organisées en 6 catégories |
| Zones de police | 3 indépendantes + police par élément |
| Résolutions | 6 presets + dimensions personnalisées |
| Nations intégrées | 31 avec drapeaux SVG |
| Formats d'import roster | CSV, Excel, JSON |
| Formats d'export | PNG, PDF, WebM, MP4, GIF, JSON, NDJSON |
| Overlays simultanées | jusqu'à 10 (Lower Third, Score Bug, Ticker) |
| Niveaux d'undo | 50 |
| Éléments de bibliothèque | 45 dans 11 catégories |
| Compatibilité broadcast | OBS, vMix, Wirecast, CasparCG, Vizrt |
| Fonctionnement | 100 % navigateur, zéro installation |
